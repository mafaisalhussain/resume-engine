# AI Fitness Rep Counter

## Domain
Computer Vision | Edge AI | AI Fitness Systems | Real-Time Pose Estimation

## Technologies
Python, OpenCV, MediaPipe, NumPy, NVIDIA Jetson Nano 4GB, HTML, CSS, JavaScript, HTTPServer

## Concepts
Pose Detection, Edge AI Computing, Real-Time Computer Vision, Motion Tracking, Angle Calculations, Workout Analytics, Gamification Systems

## Detailed Work

- Built real-time AI-powered workout tracker using webcam-based pose estimation
- Deployed and optimized computer vision workflows on NVIDIA Jetson Nano 4GB edge device
- Implemented MediaPipe Pose Detection for live body landmark tracking
- Developed automatic exercise rep counting using joint angle analysis
- Engineered low-latency real-time inference pipeline for embedded AI systems
- Designed modular exercise detection architecture for multiple workout types (Bicep Curl, Squat, Shoulder Press)
- Created live workout HUD overlay with exercise metrics and visual feedback
- Engineered gamification system with XP progression, streak tracking, and level systems
- Implemented tempo analysis system for workout pacing classification
- Built personal best tracking and session analytics storage using JSON
- Developed lightweight browser-based analytics dashboard via HTTPServer

## Architecture Highlights

- Webcam frame acquisition pipeline
- Real-time MediaPipe pose landmark extraction
- Embedded inference optimization on Jetson Nano 4GB
- Joint angle computation engine
- Rep state transition logic (finite state machine)
- Gamification and progression system
- Session persistence layer (JSON)
- Local analytics dashboard
- Modular exercise configuration architecture
