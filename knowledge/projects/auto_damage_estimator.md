# Auto Damage Estimator

## Domain
Big Data Engineering | Machine Learning | Computer Vision | Distributed AI Systems

## Technologies
Python, PySpark, Apache Spark, TensorFlow, EfficientNet-B0, Gradio, Joblib, Pandas, NumPy

## Concepts
Distributed Data Processing, Feature Engineering, Deep Learning Embeddings, Spark ML Pipelines, Computer Vision, Large-Scale Data Processing

## Detailed Work

- Built scalable Big Data pipeline for vehicle damage detection and repair cost estimation
- Developed distributed image processing workflows using Apache Spark and PySpark
- Implemented large-scale COCO-format dataset ingestion and preprocessing pipeline
- Engineered distributed feature extraction system using EfficientNet-B0 embeddings
- Designed Spark DataFrame transformation workflows for structured image analytics
- Built automated data cleaning and validation pipelines for corrupt and incomplete annotations
- Implemented parallel image loading and distributed preprocessing architecture
- Developed feature vector storage system using partitioned parquet files
- Created Spark ML pipeline integrating VectorAssembler and classification workflows
- Implemented KNN-based secondary prediction system for damage classification
- Built machine learning pipeline for repair cost estimation
- Developed interactive Gradio-based web interface for inference and predictions
- Engineered end-to-end workflow from dataset ingestion to deployment
- Optimized distributed computation workflows for scalable image analytics
- Designed modular architecture separating ingestion, feature engineering, training, and deployment
