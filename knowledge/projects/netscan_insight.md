# NetScan-Insight

## Domain
Cybersecurity | Network Monitoring | Machine Learning | Anomaly Detection Systems

## Technologies
Python, Scapy, Streamlit, Scikit-learn, Isolation Forest, Pandas, NumPy

## Concepts
Network Traffic Analysis, Packet Sniffing, Anomaly Detection, Unsupervised Learning, Real-Time Monitoring, Feature Extraction

## Detailed Work

- Built real-time network traffic monitoring and anomaly detection system
- Implemented live packet sniffing pipeline using Scapy
- Developed feature extraction workflows from raw packet metadata
- Engineered anomaly detection system using Isolation Forest unsupervised learning model
- Designed lightweight machine learning pipeline for suspicious traffic identification
- Built Streamlit dashboard for interactive network monitoring and visualization
- Implemented packet parsing workflows for IP, protocol, and packet length analysis
- Developed real-time traffic analytics architecture for continuous monitoring
- Created modular pipeline separating packet capture, feature extraction, and anomaly analysis
- Optimized lightweight detection workflows for local execution environments

## Detection Pipeline

### Packet Capture
- Live packet sniffing via Scapy, real-time ingestion, protocol-level analysis

### Feature Engineering
- IP extraction, protocol analysis, packet length metrics, traffic metadata processing

### Machine Learning
- Isolation Forest anomaly detection, unsupervised behavior analysis, suspicious traffic identification

### Visualization
- Streamlit dashboard, interactive anomaly inspection, real-time traffic monitoring

## Architecture Highlights

- Real-time packet ingestion pipeline
- Network feature extraction layer
- Isolation Forest inference engine
- Traffic anomaly scoring system
- Interactive Streamlit monitoring dashboard
