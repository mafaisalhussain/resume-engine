# Resume Section Order

The following section order is mandatory for all resumes. Never deviate from this order regardless of JD or role type.

1. Professional Summary
2. Projects
3. Experience
4. Skills
5. Education
6. Certifications

**Rules:**
- Projects must appear immediately after Professional Summary and before Experience.
- Experience must appear before Skills.
- Skills must appear before Education.
- Education must appear before Certifications.
- If a section has no content (e.g., no certifications), omit it entirely — do not include an empty section.
- This order applies to both PATH 1 and PATH 2 role types with no exceptions.
